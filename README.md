# kde-config-for-fedora
This is my kde configaration for fedora on my thinkpad T490 so i dont need to configure themes, shortcut, some apps, kde settings again. But apps are no include. So next time i install kde fedora i use this then restart and install apps, but be carefull because maybe future kde or fedora updates will break something 

the way to use it on fedora KDE :

1- make sure git is installed "sudo dnf install git"

2- type this command "sudo git clone https://github.com/mklx25/kde-config-for-fedora.git"

3- enter the kde-config-for-fedora folder "cd kde-config-for-fedora"

4- make setup.sh executable "chmod +x setup.sh"

5-run the setup.sh script "./setup.sh"
